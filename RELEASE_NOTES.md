# Release Notes

## 2026-07-24

### Highlights
- Added history score entry for each court, with automatic save.
- Default score data is stored as 0-0 while keeping the input fields empty until edited.
- History view can be filtered by member number.
- Filtered history also shows only the courts involving the selected member.
- Filtered history hides rest information for a cleaner view.
- Added member pair history and opponent history statistics.
- Added individual win ranking display.
- Updated pairing logic to prefer unseen pairs using accumulated pair history.

### Notes
- Individual win ranking no longer shows win rate.
- Score inputs are limited to two digits.
- Empty score fields are preserved visually for easier editing.
- Member-filtered history keeps match rounds and member-change entries together when relevant.
