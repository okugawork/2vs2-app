# ピックル試合

2vs2向けの試合作成を行う Web アプリです。人数とコート数を選択し、休憩を含めた次試合の組み合わせを作成できます。

## 最新リリース

- 2026-07-24: 履歴の点数入力、自動保存、メンバー別履歴フィルター、勝利数統計を追加
- 詳細は [RELEASE_NOTES.md](RELEASE_NOTES.md) を参照

## 技術構成

- HTML
- CSS
- JavaScript (Vanilla)

## 現在の主な機能

- 人数とコート数を選択して試合作成
- コートごとのペア表示 (コート1, コート2...)
- 休憩中メンバー表示
- 休憩回数テーブル表示
- ペア履歴をもとに未ペアを優先する組み合わせロジック
- 休憩回数が少ないメンバーを優先した休憩選出
- 途中参加 (メンバー追加)
- 途中退場 (メンバー除外)
- 追加/除外の保留メッセージ表示
- 参加人数横の差分バッジ表示 (+n / -n)
- メンバー別ペア履歴の統計表示
- メンバー別対戦相手履歴の統計表示
- 履歴モードからの点数入力と勝敗記録
- 勝利数・勝率ランキング表示
- 試合履歴タブ
- メンバー変更履歴タブ内表示
- リセット機能

## 画面と動作

### 1. セットアップ画面

- 参加人数とコート数を選択
- ペア作成を押すと確認モーダルが表示され、作成で試合生成

### 2. 試合進行画面

- 作成された試合を MATCHBOARD に表示
- 休憩中メンバーを MATCHBOARD 下部に表示
- メンバー追加/除外が可能
   - 追加時: 次試合から参加
   - 除外時: 次試合から除外
- 保留メッセージは複数行で表示
- 参加人数右側に差分バッジを表示
   - 追加: 緑の +n
   - 除外: 赤の -n

### 3. 履歴タブ

- 最新が上に来る並び順
- 試合作成1回を 第n試合 として保存
- 第n試合カードの見出しに開始時刻を表示
   - 表示形式: yyyy/mm/dd hh:mm:ss
- 複数コート利用時は、同一試合カード内にコート別枠を表示
- 各コートごとに点数を入力でき、保存すると勝敗を記録
- 保存済みの点数に応じて、各コートの勝者を表示
- メンバー追加/除外の履歴も表示
   - 時刻は表示しない
   - 同じ反映回 (第n試合から反映) の変更は1カードに集約

## 組み合わせロジック

- 1回の作成で作る試合数
   - min(コート数, floor(参加人数 / 4))
- 余剰人数は休憩
- 休憩は休憩回数が少ないメンバーを優先
- ペア生成は累積のペア履歴をもとに、未ペアを優先しつつ選定

## 既知の仕様メモ

- ペア作成ボタンを押した時点で、追加/除外の保留メッセージはクリアされます。
   - 試合が0件だった場合でもクリアされます。
- 対戦相手履歴を使った試合ペア選定の最適化は今後のイシューとして残しています。
