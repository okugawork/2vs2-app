# ピックル試合

## 目的

2vs2スポーツ向けのペア組み合わせ作成アプリを、GitHub Pages上で公開するための個人開発プロジェクトです。

## 初期構成

- HTML
- CSS
- JavaScript（Vanilla JS）
- Git / GitHub
- GitHub Pages

## Ver1 機能

- 名前入力欄
- 登録ボタン
- 参加者一覧表示

## ファイル構成

- `index.html`
- `styles.css`
- `script.js`

## 実行方法

1. このフォルダをGitHub Pagesの公開ルートに配置します。
2. `index.html` をブラウザで開きます。

---

今後は Ver2 で参加者削除機能、Ver3 で組み合わせ生成機能を追加します。
