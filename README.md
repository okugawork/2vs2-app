# ピックル試合

## 目的

2vs2スポーツ向けのペア組み合わせ作成アプリを、GitHub Pages上で公開するための個人開発プロジェクトです。

## 初期構成

- HTML
- CSS
- JavaScript（Vanilla JS）
- Git / GitHub
- GitHub Pages

## 現在の機能

- 人数選択
- ペア作成ボタン
- ランダムな2vs2組み合わせ生成
- 余りの人数表示

## 仕様

- 4人以上の人数を選択すると、2vs2の試合を作成できます。
- 12人なら3試合の6ペアを作成します。
- 14人なら3試合の6ペアで2人余ります。
- 余りは「余り: 5, 12」のように表示します。

## ファイル構成

- index.html
- styles.css
- script.js

## 実行方法

1. このフォルダをGitHub Pagesの公開ルートに配置します。
2. index.html をブラウザで開きます。
